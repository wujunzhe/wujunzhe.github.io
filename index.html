canvas学习

1.HTML部分

​ <canvas id="canvas" width="300" height="300">您的浏览器不支持canvas</canvas>

在HTML页面添加一个canvas标签，设置好id、宽、高。id是为了方便js获取元素。

  • 每个canvas初始都是一个白板;
  • 样式除了宽高，尽量写在画布里而不是css;
  • canvas标签里如果有文本，当浏览器不支持canvas时会显示文本;

2.绘图上下文环境

  • 获取canvas元素;
  • 调用getContext()方法获取绘图环境上下文，参数为2d;

绘图上下文包含所有绘制方法和属性的定义

3.路径

3.1 beginPath( ) 开始新路径

表示重新开始的路径。
canvas是基于状态的绘图，开启一条新路径，就可以设置新的状态。 但是beginPath( ) 不会将状态重置回默认值

同一条路径上的线条和曲线都会是相同的颜色。 所以需要绘制不同颜色的时候，必须开启新的路径。

3.2 closePath( ) 结束路径

用于结束路径。与 beginPath( )并不一定成对出现。

tips: 路径结束后，图形会自动封闭。所以有时候用不到结束路径

3.3 stroke( ) 绘制路径

绘制已经定义好的路径

3.4 fill( ) 填充路径

将准备画的路径块填充

fill( )会覆盖描边stroke( )边框的一半。所以，在需要描边的填充色，需要先填充，后描边。

3.5 isPointInPath( ) 是否在路径中

如果指定的点位于当前路径中，返回true，不在的话返回false。

context.isPointInPath(x,y);

  • x :点的x坐标;
  • y :点的y坐标

3.6 clip( ) 剪切

从原始画布剪切任意形状的尺寸和区域

context.clip();

注意 : 一旦剪切了某个区域，则所有之后的绘图都会被限制在被剪切的区域内（不能访问画布上的其他区域）。您也可以在使用 clip( ) 方法前通过使用 save( ) 方法对当前画布区域进行保存，并在以后的任意时间通过 restore( ) 方法进行恢复。

4、线条以及填充颜色

4.1 fillStyle填充样式

设置或返回用于填充绘画的颜色、渐变或模式，需要在填充前声明才有效

context.fillStyle = color | gradient | pattern;

  • color 指示绘图填充色的CSS颜色值。默认是 #000000。
  • gradient 用于填充绘图的渐变对象
  • pattern 用于填充绘图的 pattern 对象

4.2 strokeStyle 线条样式

设置或返回线条的颜色、渐变或模式，同样需要在绘制前声明才有效

context.strokeStyle = color | gradient | pattern;

  • color 指示线条颜色的CSS颜色值。默认是 #000000。
  • gradient 用于填充绘图的渐变对象
  • pattern 用于创建线条的 pattern 对象

4.3 lineCap 线条结束端点的样式

设置或返回线条结束端点的样式

context.lineCap = ' butt | round | square';

  • butt 默认值。向线条的每个末端添加平直的边缘。
  • round 向线条的每个末端添加圆形线帽。
  • square 向线条的每个末端添加正方形线帽。

round 和 square 会使线条略微变长。

4.4 lineJoin 线条拐角类型

设置或返回两条线相交时，所创建的拐角类型

context.lineJoin = " bevel | round | miter";

  • bevel 创建斜角。
  • round 创建圆角。
  • miter 默认值。创建尖角。

4.5 lineWidth( ) 线条宽度

设置或返回当前线条的宽度

context.lineWidth = number

  • number 当前线条的宽度，单位为px。

4.6 miterLimit 线条的最大斜接长度

设置或返回最大斜接长度

context.miterLimit = number

  • number 正数
  • 如果斜接长度大于 miterLimit 的值，边角会以 lineJoin 的 bevel 显示

斜接长度指的是在两条线的交汇处内角与外角之间的距离。
